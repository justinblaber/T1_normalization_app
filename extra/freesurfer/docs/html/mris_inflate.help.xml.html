NAME

mris_inflate - inflate cortical surface

SYNOPSIS

mris_inflate [<options>] insurfacefile outsurfacefile

DESCRIPTION

This program will inflate a cortical surface.

POSITIONAL ARGUMENTS

ArgumentExplanation
insurfacefileinput surface file
outsurfacefileoutput surface file

REQUIRED-FLAGGED ARGUMENTS

None

OPTIONAL-FLAGGED ARGUMENTS

ArgumentExplanation
-n <# of iterations>set the maximum # of iterations (default = 10)
-w <# of iterations>write out a snapshot of the inflation every #th time step.
-dist <distance coefficient>specify the relative strength of the metric preserving term in the cost functional versus the smoothing term (default = 0.100000).
-no-save-sulcdo not save ?h.sulc
-sulc sulcnamesave to ?h.sulcname

OUTPUTS

OutputExplanation
outsurfacefileoutput surface file
