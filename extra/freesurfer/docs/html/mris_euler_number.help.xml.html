NAME

mris_euler_number

SYNOPSIS

mris_euler_number [<options>] insurf

DESCRIPTION

This program computes EulerNumber for a cortical surface

POSITIONAL ARGUMENTS

ArgumentExplanation
insurfinput surface file

REQUIRED-FLAGGED ARGUMENTS

None

OPTIONAL-FLAGGED ARGUMENTS

-o outfile

ArgumentExplanation
  write number of holes to outfile

OUTPUTS

OutputExplanation
eulernumberEulerNumber as text to stdout

EXAMPLE 1

mris_euler_number surface
