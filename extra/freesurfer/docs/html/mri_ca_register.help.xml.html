NAME

mri_ca_register

SYNOPSIS

mri_ca_register <invol> <template> <outvol>

DESCRIPTION

This generates a multi-dimensional talairach transform from a gca file and talairach.lta file

POSITIONAL ARGUMENTS

ArgumentExplanation
involinput volume
templatetemplate in gca format
outvoloutput volume in m3d format

REQUIRED-FLAGGED ARGUMENTS

*******************************************************************************************

OPTIONAL-FLAGGED ARGUMENTS

ArgumentExplanation
-toldefines convergence of registration
-maskspecifies volume to use as a mask
-Ttransform in lta format
-leveldefines how many surrounding voxels will be used in interpolations, default is 6
-riallows reading of multiple intensity normalization
-align 
-invert-and-save gcamfile 
-dist distance 
-regularize regularize 
-regularize-mean regularizemean 
-scale_smoothness smoothness 
-nobright  
-renormalize 
-read_lta  
-smoothness smoothness 
-samples samples 
-nsmall nsmall 
-fixed 
-optimal 
-noneg noneg 
-wm 
-min_avgs <n> 
-tl 
-relabel relable 
-relabel_avgs relable_avgs 
-reset_avgs reset_avgs 
-vf vf_fname 
-diag diagfname 
-tr tr 
-te te 
-example T1 seg 
-<no>bigventricles 
-uncompress 
-secondpassrenorm

OUTPUTS

OutputExplanation
outvolmultidimensional transform in m3d format

EXAMPLE 1

mri_ca_register -levels 6 -mask subject1/mri/brain -T subject1/mri/transforms/talairach_one.lta subject1/mri/norm_one.mgh single_one.gca subject1/mri/transforms/talairach_one.m3d

EXAMPLE 2

mri_ca_register -tol 1 -mask subject1/mri/brain -T subject1/mri/transforms/talairach_multi.lta subject1/mri/norm_multi.mgh multi_one.gca subject1/mri/transforms/talairach_multi.m3d
