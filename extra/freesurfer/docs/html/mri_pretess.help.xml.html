NAME

mri_pretess

SYNOPSIS

mri_pretess [options] filledvol labelstring normvol newfilledvol

DESCRIPTION

Changes white matter (WM) segmentation so that the neighbors of all voxels labeled as WM have a face in common - no edges or corners allowed.

POSITIONAL ARGUMENTS

ArgumentExplanation
filledvolusually wm.mgz
labelstringusually wm
normvolusually norm.mgz
newfilledvolusually wm.mgz

OPTIONAL-FLAGGED ARGUMENTS

ArgumentExplanation
-debug_voxel C R S 
-nocornersno removal of corner configurations in addition to edge ones
-wturn on diagnostic writing
-keepkeep WM edits
-testadds a voxel that should be removed by mri_pretess. The value of the voxel is set to that of an ON-edited WM, so it should be kept with -keep. The output will NOT be saved.
